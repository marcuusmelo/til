# SQL Terminology

## Data Definition Language (DDL)
- create table,
- drop table, 
- and other creates and drops of other components.

## Data Manipulation Language (DML)
- select, 
- insert, 
- delete, 
- update. 
Querying and modifying the database.


## Other Commands
- indexes,
- constraints,
- views,
- triggers,
- transactions,
- authorization,
- etc.
